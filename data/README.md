# Data Availability

## Innovation Fuel Podcast Dataset

The Innovation Fuel podcast is produced by University Canada West and is publicly available at:

🎙️ **https://www.ucanwest.ca/innovation-fuel**

The dataset used in this study comprises **124 episodes** in which venture founders narrate the complete trajectory of their ventures. All transcripts were obtained from the publicly available podcast feed.

---

## Corpus Statistics

| Metric | Value |
|--------|-------|
| Total episodes | 124 |
| Episodes with strategy labels | 121 |
| Episodes excluded (zero strategy labels) | 3 |
| Total sentences (after preprocessing) | 38,508 |
| Mean sentences per episode | 310 |
| SD | 142 |
| Range | 78–982 |
| Strategy label instances | 666 across 658 sentences |
| Constraint label instances | 5,450 across 5,450 sentences |
| Sentences with both label types | 162 (0.42%) |

---

## Why Raw Data Is Not in This Repository

The episode transcripts are not included in this repository because:
1. They are publicly available from the original source
2. Reproducing them here would be redundant given their open availability
3. File size (38,508 sentences × full text)

## Intermediate Output Files

All intermediate analysis outputs are **deterministic functions** of the input corpus and the published code. They are not separately deposited to avoid redundancy.

Researchers who need specific intermediate files may either:
- Run `notebooks/Paper4_Network_Unified.ipynb` (completes in ≈ 8 minutes)
- Contact the corresponding author: farhadiangelareh@gmail.com

### Output files generated by the notebook

| File | Description |
|------|-------------|
| `sentences.csv` | All 38,508 sentences with episode_id, sentence_id, sent_idx, text |
| `labels.csv` | 666 strategy label instances with episode_id, sentence_id, strategy |
| `episode_strategy_counts.csv` | Raw strategy label counts per episode |
| `episode_strategy_share.csv` | Row-normalized strategy share vectors per episode |
| `episode_clusters.csv` | Cluster assignment for each episode |
| `transition_matrix.csv` | 10×10 row-normalized transition probability matrix |
| `transition_counts.csv` | 10×10 raw transition count matrix |
| `top_3step_paths.csv` | Top three-step strategy path sequences |
| `co_matrix.csv` | 10×10 strategy co-occurrence matrix |
| `episode_similarity.csv` | Pairwise cosine similarity matrix (121×121) |
| `centrality.csv` | Node centrality metrics for episode network |
| `validation_metrics.csv` | Robustness analysis results |
| `labels.csv` | Strategy label instances |
